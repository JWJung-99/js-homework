# 엘리멘탈 포스터 제작

다음 과제 : 포스터 변경 시 재생 중이던 오디오 정지...

<br><br>

## 1. 프로젝트 개요
<br>
썸네일 이미지를 클릭하면 메인 이미지와 배경이 변경되는 포스터를 제작합니다.

<br><br>

## 2. 프로젝트 정보
<br>

- 개발 인원 : 1명
- 개발 기간 : 24/02/09 ~ 24/02/09 (약 1일 간)

<br><br>

## 3. 프로젝트 설명
<br>

- 이벤트 위임 방식을 활용한 이벤트 핸들링
- 내비게이션에서 이미지를 클릭 시
   - 배경 색상, 메인 이미지, 메인 텍스트 변경
   - 선택한 이미지 강조
- 함수 분리 및 모듈 프로그래밍
- 각 요소 별 오디오 추가

<br><br>

## 4. 주요 기능

- 엠버 포스터

- 웨이드 포스터

- 클로드 포스터

- 게일 포스터
